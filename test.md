##### **（4）交互与技术特性**

- **前端框架**：基于 Next.js 构建（由 `__NEXT_DATA__` 与 `next-route-announcer` 等脚本判断）。
- **动态加载**：门店数据通过 `__NEXT_DATA__` JSON 字符串注入，实现服务端渲染（SSR）与客户端交互。
- **可访问性支持**：
  - 使用 `aria-label`、`aria-expanded`、`aria-live` 等 ARIA 属性，支持屏幕阅读器。
  - 按钮与链接具备清晰的语义化标签。
- **响应式设计**：适配移动端与桌面端，支持触控操作（如折叠面板）。
- **地图集成**：使用 `mapkit.js`（Apple MapKit）实现地图功能，支持门店定位与导航。
- **无障碍优化**：支持键盘导航、焦点管理、动态内容更新提示。
